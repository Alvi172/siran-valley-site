# Siran Option A GitHub Upload Guide

1. Download and unzip `siran_option_a_seo_domination_full_site.zip`.
2. Open your GitHub repository for siran.pk.
3. Upload everything from inside the unzipped folder into the repository root.
4. Keep the `/blog/` folder exactly as delivered.
5. Replace old `index.html` with the new `index.html`.
6. Upload `sitemap.xml`, `robots.txt`, `site.webmanifest`, favicon files and all image assets.
7. Commit message: `Deploy Siran multi-page SEO system`.
8. Wait 2–5 minutes, then open https://www.siran.pk and hard refresh.
9. In Google Search Console, submit: https://www.siran.pk/sitemap.xml

Do not delete old PNGs until the new site is fully checked.
